# The Sudoku Model

A sudoku game consists of a {ref}`Field <field>` with 81 {ref}`Cells <cell>`. 9 cells per row, and 9 cells per column. It has also 9 {ref}`groups or blocks <block>` that have 3 rows and columns each.
